
index.php
	=> page par defaut du site
	
functions.php
	=> les diverses fonctions utilisees dans le site
	
boilerplate.php
	=> configuration du site, en gros c'est la partie <head> du html
	
config.php
	=> definition des varibles globales pour configurer le titre / la db par exemple

install_db.php
	=> crée les tables dans la base de données online_bet.
	- Pour installer la BDD, créer la BDD online_bet sur phpmyadmin, configurer les variables dans config.php et se connecter à install_db.php depuis son navigateur.
